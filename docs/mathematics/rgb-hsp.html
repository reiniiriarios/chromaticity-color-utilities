RGB and HSP

Red, Green, Blue ⇄ Hue, Saturation, Perceived Brightness

RGB to HSP​

Where P is perceived brightness. PR, PG, and PB are weights that can be adjusted as you will, such that PR + PG + PB= 1

If no Pi values are passed to the method, the default weight for P is as follows:

PR=0.299PG=0.587PB=0.114\begin{align*} P_{R} &= 0.299 \\ P_{G} &= 0.587 \\ P_{B} &= 0.114 \end{align*}PR​PG​PB​​=0.299=0.587=0.114​

This algorithm is similar to the one Photoshop uses when converting images to greyscale.
